- [**I Cut My Cloud Bill with Tailscale**](https://youtu.be/Wh_HpPwSswo) [Video, English]
  Twitter user @adriandotgoins shares how he cut his cloud bills by 95% with Tailscale. [Part One](https://www.youtube.com/watch?v=Wh_HpPwSswo) | [Part Two](https://www.youtube.com/watch?v=OIwxIdyZg7A)

- [**Use Tailscale For SSH Access To "Lan" Locked Machines**](https://www.youtube.com/watch?v=JHuD4Ab45bk) [Video, English]
  This video demonstrates how to use Tailscale to access web development machines from anywhere. No static IPs, no dyndns, no port forwarding required.

- [**Tailscale and Docker**](https://gitlab.com/gruberx/tailscale-docker)
  Gitlab user [Gruber](https://gitlab.com/gruberx) shares their Tailscale Docker-based relay.

- [**Replace personal VPN with Tailscale**](https://blog.gslin.org/archives/2021/12/29/10473/%e7%94%a8-tailscale-%e5%8f%96%e4%bb%a3%e5%80%8b%e4%ba%ba%e7%9a%84-vpn/) [Chinese]
  Gea-Suan Lin gives a review of Tailscale.

- [**Using Tailscale to remotely access Synology and Vaultwarden**](https://jussiroine.com/2021/12/building-a-more-secure-remote-access-setup-with-tailscale-to-access-vaultwarden-and-synology-remotely/)
  [Jussi Roine](https://www.linkedin.com/in/jroine/) shares how they built a more secure setup with Tailscale to access Vaultwarden and Synology.

- [**Tailscale and WireGuard**](https://github.com/corelight/zeek-spicy-wireguard)
  Twitter user [@keithjjones](https://mobile.twitter.com/keithjjones) shares a Zeek WireGuard protocol analyzer based on Spicy.

- [**Tailscale for Mikrotik Containers**](https://github.com/Fluent-networks/tailscale-mikrotik)
  Build and configuration information to run Tailscale in Mikrotik Containers.

- [**Tailscale and Cloudfare**](https://github.com/marc1307/tailscale-cloudflare-dnssync)
  GitHub community member [@marc1307](https://github.com/marc1307) shows how to Sync Tailscale host IPs to a Cloudflare hosted DNS.

- [**How To Get Tailscale Working With a Fire TV Stick**](https://www.reddit.com/r/fireTV/comments/r5fiom/guide_on_how_to_get_tailscale_working_with_a_fire/)
  Reddit user “Broke Tech” shares how to get Tailscale working with a Fire TV Stick.

- [**Configurable terraform for a free, firewalled, oracle-cloud ARM Ubuntu machine**](https://github.com/stealthybox/tf-oci-arm)
  [Leigh Capili](https://github.com/stealthybox) shares configurable terraform for a free, firewalled, Oracle-cloud ARM Ubuntu machine with Tailscale and Docker.

- [**Connect your Kubernetes Cluster with Tailscale**](https://github.com/adamgoose/tsk)
  [Adam Engebretson](https://github.com/adamgoose) shows how to quickly connect a Kubernetes Cluster with Tailscale.

- [**Local reverse proxies for Tailscale on WSL2**](https://github.com/dhermes/tailscale-wsl2)
  [Danny Hermes](https://twitter.com/bossylobster) showcases their GitHub feature on a pair of local reverse proxies (one in Windows, one in Linux) for Tailscale on WSL2.

- [**Gitpod Template with Tailscale**](https://mobile.twitter.com/gbraad/status/1467752150315188230)
  [Gerard Braad](https://mobile.twitter.com/gbraad) created a Gitpod template with Fedora and Tailscale.

- [**Super-Charged Remote Development with Tailscale and Jetbrains Projector**](https://mobile.twitter.com/Artis3n/status/1468609844450144268)
  Ari Kalfus wanted to easily stand up a remote dev server they could connect to from a browser and keep private and secure. Tailscale helped achieve this.

- [**Bare Metal Kubernetes with Tailscale**](https://blog.bossylobster.com/2021/12/tailsk8s.html)
  Danny Hermes shares how to get Tailscale working with a Bare Metal Kubernetes cluster. They write, “I've had a ton of fun toying with Tailscale recently (it's incredible how literally everything ‘just works’).”
