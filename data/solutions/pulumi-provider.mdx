[Pulumi][pulumi] is an infrastructure as code tool that allows you to deploy infrastructure programmatically.

The [Tailscale Pulumi provider][pulumi-provider] is maintained and published in the Pulumi registry by Pulumi.

### Features

Use the Pulumi provider for Tailscale to:

- Define your [tailnet policy file][acls], using the [`Acl` resource][acl-resource]
- Set [DNS settings][dns], including:
  - global nameservers, using the [`DnsNameservers` resource][dns-nameservers-resource]
  - restricted nameservers for split DNS, using the [`DnsSearchPaths` resource][dns-search-paths-resource]
  - enabling or disabling [MagicDNS][magicdns], using the
    [`DnsPreferences` resource][dns-preferences-resource]
- Generate an [auth key][auth-key], including setting whether itâ€™s reusable, ephemeral, pre-authorized, and tagged,
  using the [`TailnetKey` resource][key-resource]
- Manage properties of a device, including:
  - approving the device, using the [`DeviceAuthorization` resource][device-authorization-resource]
  - disabling key expiry, using the [`DeviceKey` resource][device-key-resource]
  - setting tags, using the [`DeviceTags` resource][device-tags-resource]
  - advertising subnet routes, using the [`DeviceSubnetRoutes` resource][device-subnet-routes-resource]

### Installation steps

To use Pulumi with Tailscale:

1. Install the package for the Tailscale Pulumi provider in [Node.js, Python, Go, or .NET][package].

1. Set the Tailscale configuration for Pulumi with an [API access token](/kb/1101/api/) for Tailscale and with the name of your tailnet. You can either
   [set these as environment variables or as part of your Pulumi configuration][config-options]. To set these in your Pulumi configuration:

   ```shell
   pulumi config set tailscale:apiKey tskey-1234567CNTRL-abcdefghijklmnopqrstu --secret
   pulumi config set tailscale:tailnet example.com
   ```
